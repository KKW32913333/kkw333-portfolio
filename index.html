Linkle

公開中

家族・友人グループで予定・タスク・買い物リスト・勤務表を共有できるWebアプリです。カレンダーによるスケジュール管理、買い物・ToDoリストの共同管理、勤務表のシフト集計、グループチャットや掲示板機能も搭載。プッシュ通知とPWA対応で、スマホでもアプリのように使えます。

Shelfy

公開中

棚（Shelf）を起点に、買いすぎ・食品ロス・使い忘れをなくす在庫管理アプリです。賞味期限のカラー表示や在庫アラート、写真付きのアイテム管理に加え、在庫が0になると自動でLinkleの買い物リストに追加される連携機能も搭載。消費ログや月次レポート、スコア・バッジ機能でゲーム感覚で在庫管理を続けられます。

📷 スクリーンショット準備中
Java 17 Spring Boot Thymeleaf PostgreSQL Cloudflare R2 PWA
GitHub 本番サイトを見る

Finterra

公開中

個人・グループどちらにも対応した家計簿・資産管理アプリです。取引の登録・検索に加え、投資信託や株式・預金などの資産推移をグラフで可視化。予算超過や固定費の自動計上を通知で知らせ、レシート画像のOCR読み取りにも対応しています。
